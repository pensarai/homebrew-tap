# Homebrew Tap for Apex

This is the official Homebrew tap for [Apex](https://github.com/pensarai/apex), an AI-powered penetration testing CLI tool by Pensar.

## Installation

```bash
brew tap pensarai/apex
brew install apex
```

## Usage

```bash
# Launch the TUI
pensar

# Show help
pensar --help

# Show version
pensar --version
```

## Updating

```bash
brew update
brew upgrade apex
```

## Uninstalling

```bash
brew uninstall apex
brew untap pensarai/apex
```
